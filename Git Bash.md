### 사용자 인증정보 설정
```
$ git config --global user.name "John Doe"
$ git config --global user.email sabarada@naver.com
```

```
git remote add origin https://github.com/aldrn29/~.git
```

```
git init
```

```
git clone https://github.com/aldrn29/~.git
```

```
git branch -M main
```
